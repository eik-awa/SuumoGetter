# 🦠 SuumoGetter

SUUMOから賃貸物件情報を自動収集し、Excelファイルに出力するスクレイピングツールです。

---

## 概要

指定した沿線・駅周辺の賃貸物件情報をSUUMOから自動取得し、平米単価順に並べたExcelファイルを生成します。<br>
物件探しの効率化や、家賃相場の把握に役立てることができます。

---

## 取得できるデータ

| 項目 | 説明 |
|------|------|
| 平米単価 | (賃料 + 管理費) ÷ 専有面積 |
| 駅まで | 最寄り駅からの所要時間 |
| 築年数 | 建築からの年数（新築は0） |
| 階数 | 部屋の所在階 |
| 賃料 | 月額賃料（円） |
| 管理費 | 月額管理費（円） |
| 敷金 | 敷金（円） |
| 礼金 | 礼金（円） |
| 専有面積 | 部屋の広さ（㎡） |
| 間取り | 1K / 2LDK など |
| リンク | SUUMO物件詳細ページへのリンク |

---

## 必要環境

- Python 3.x
- Google Chrome
- ChromeDriver（Chromeのバージョンと一致したもの）

## 必要ライブラリ

```bash
pip install selenium chromedriver-binary openpyxl
```

---

## 使い方

`main()` に沿線名・駅名・取得上限件数を渡して実行します。

```python
if __name__ == "__main__":
    sg = SuumoGetter()
    sg.main("ＪＲ中央線", "吉祥寺", 100)
```

### パラメータ

| 引数 | 型 | 説明 | 例 |
|------|----|------|----|
| `line` | str | 沿線名（全角） | `"ＪＲ山手線"` |
| `station` | str | 駅名 | `"秋葉原"` |
| `limit` | int | 取得する物件の上限件数 | `100` |

> ⚠️ 沿線名はSUUMOサイト上の表記に合わせて**全角文字**で入力してください。

---

## 出力ファイル

実行後、カレントディレクトリに `rental.xlsx` が生成されます。

### Excelの特徴

- **平米単価の昇順**で並べ替え済み（コスパの良い物件が上位に）
- 1行目に**フィルター**が設定済み
- 1行目が**固定表示**（スクロールしても見出しが消えない）
- リンク列は**ハイパーリンク**として設定（クリックで物件ページへ遷移）

## クラス構成

```
SuumoGetter
├── __init__()         # ChromeDriverの初期化
├── click()            # 安定クリック処理（画面外要素にも対応）
├── rental_search()    # 沿線・駅名でSUUMOを検索
├── rental_get()       # 物件情報を取得してリストで返す
├── export_excel()     # 取得データをExcelに出力
└── main()             # 上記を統合したメイン関数
```

---

## 注意事項

- 本ツールはSUUMOのWebページ構造に依存しています。サイト側の仕様変更により動作しなくなる場合があります。
- スクレイピングは対象サービスの利用規約を確認のうえ、個人的な用途に限ってご利用ください。
- 取得件数が多い場合は時間がかかることがあります。`limit` パラメータで件数を絞ることを推奨します。
